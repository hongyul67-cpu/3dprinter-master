# 🖨️ 3D프린터운용기능사 필기 마스터

3D프린터운용기능사 **필기** 대비 학습게임입니다. 개념 학습과 기출 문제(약 92문항),
오답노트를 게임처럼 즐기며 공부합니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/3dprinter-master/

## 주요 기능
- **개념 학습** + **기출 92문항** 풀이
- **오답노트** — 틀린 문제 집중 복습
- 게임형 학습 진행

## 결과수집 (교사용)
학생 결과를 교사 구글시트로 모을 수 있습니다.
- `AppsScript_코드.gs` — 구글 시트 수집용 Apps Script
- `교사용_구글시트_설치법.txt` — 설치 안내 (약 5분)
- `config.js` — 결과수집 주소 등 설정

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. 설치가 필요 없습니다.
